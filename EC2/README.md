# 🖥️ Amazon EC2

## What is EC2?

Amazon EC2 (Elastic Compute Cloud) provides scalable virtual servers in the cloud.

---

## Features

- Virtual Machines
- Scalable Compute
- Multiple Operating Systems
- Security Groups
- SSH Access

---

## Common Use Cases

- Web Hosting
- Application Servers
- Development & Testing

---

## Learning Outcome

Understand how to launch and manage EC2 instances.
